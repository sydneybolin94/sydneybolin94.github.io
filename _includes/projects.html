Modular Page Builder - React/Tailwind

  • Recursive component rendering
  • JSON schema validation and guardrails
  • Theme system architecture (React + Tailwind)
  • Live preview with real-time state updates
  • UX design for clarity and safe editing

This project is a fully modular page builder that lets users visually assemble layouts by editing JSON directly in an embedded code editor. The renderer reads the JSON recursively and maps each block to a real React/Tailwind component, updating the preview instantly as users type. A custom schema validation layer provides guardrails that prevent invalid structures and guide users toward safe, predictable edits. The theme system allows layouts to adapt to different visual styles without changing component logic.

Western Nebraska Website - WordPress Custom Theme + Plugin

  • Custom WordPress theme development (PHP templates, Advanced Custom Fields, header/footer/homepage)
  • Custom plugin architecture with taxonomies and relational data
  • Structured content entry using ACF and WordPress admin tools
  • Front-end component development for dynamic listings
  • Interactive UI patterns (scroll-aware header behavior)

This project involved building a custom WordPress theme and companion plugin for a nonprofit promoting travel across Western Nebraska. I developed the site's header, homepage, and footer, including a scroll-aware header that shifts color as users move down the page. The custom plugin powers a full directory of attractions, lodging, and regional areas using multiple taxonomies and ACF fields to structure content entry for nonprofit staff. Each business or attraction can be assigned to categories like Family Fun, Scenic Byways, or History & Culture, and the front end automatically displays listings in the correct sections. I also built the listing layouts as reusable components, even before I had the vocabulary for “shared components,” and they still hold up structurally today.
